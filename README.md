# Vocchi-Conspiracy 夜芷殺機

> 陰謀無處不在，破門殺機一觸即發。
> Conspiracy hides in the dark — the next door you open might be your last.

1. 遊戲階段與勝負條件系統
    * 玩家先在等候區域等待，按下開始後若在等候區外的話，就會當旁觀
    * 無投票模式（即沒有分白天、黑夜，但是遊戲可以選擇時間流動/指定時段）。
    * 陣營分配（殺手陣營、好人陣營、情侶陣營、歡愉陣營【搞事】，其他留待未來添加）。
    * 勝負判斷：殺手殺光好人 / 撐到時間結束勝利；好人把殺手全消滅或活到時間結束。
    * 遊戲會為每個玩家各自提供隨機任務，如吃東西、喝東西、看書、離開房間逛，睡覺，坐下，完成簡單小遊戲等。玩家可以不做，但會讓自己看不到對方拿甚麼，而且遊戲時間增加20s
    * 玩家完成後，時間不變，當下不會獲得任何負面buff
    * 簡單的遊戲計時器（倒數，但可以選擇不展示）。
    * 允許跳躍，近距離文字對話，模組包亦會附語音對話（Simple-Voice-Chat)。
    * 、默認下好人陣營有體力限制，只能奔跑一段時間，殺手則無限制。
2. 角色與職業系統
    * 用 JSON 檔案定義職業（id、陣營、被動技能、主動技能、描述）。
    * 稍後可透過指令設定池
    * 開局隨機分配職業、房間鑰匙給玩家與AI。
3. 道具與互動系統
    * 基本道具：鑰匙（只能開對應房間）、破門工具（炸彈/撬鎖）、武器（讓警察/殺手能即時攻擊）。
    * 房間門鎖機制（可破壞、臨時撬、或需要鑰匙）。
    * 槍械、煙霧彈、手榴彈、其他角色自定武器
